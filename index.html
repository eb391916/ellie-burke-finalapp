Square Grid Generator

What is This?

This is a GUI (graphical user interface) that allows you to customize the size, curve, stroke, and color of a grid of six squares. You can customize this grid to create a canvas you like, and then press the down arrow to download an .svg version of the canvas.

How Do I Use It?

  • Size: changes the dimensions of the squares
  • Curves: changes how curved or angles the corners are
  • Stroke Weight: changes the weight of the outline
  • Color: changes the fill color of the squares
  • Double Click: to change the background color
  • Press Down Arrow: to download .svg of canvas
